# Cell Simulator

## View here

https://ben-x9.github.io/cell-simulator/

## Usage instructions

Click cells to make them "alive". Click again to make them "dead".

Press "Next Generation" to generate the next state of the board.

Press "Reset" to clear the board.

## To install and run

```
npm install
npm run dev
```

## To run tests

```
npm run test
```
